            $MDP = (S, A, p(s' | s, a), r(s, a))$ \\[1mm]
            State space $S = \mathbb{R}^k$ \\[1mm]
            Action space $A = \textbf{1}$ \\[1mm]
            Transition probability $p: S \times A \times S \rightarrow [0,1]$ \\[1mm]
            Reward function $r: S \times A \rightarrow \mathbb{R}$
